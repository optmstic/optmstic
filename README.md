## Data analytics portfolio

Small, end-to-end data projects — from raw CSV to findings you can actually read. A mix of Python, R, SQL and BI, mostly built around academic or course briefs taken seriously enough to be useful.

---

### Featured projects

**[pds](https://github.com/optmstic/pds) — Flight Routes with Up to Two Stops**
Structural analysis of the global airport network (66k routes, 1,073 large airports). Lists every viable route between two airports with ≤ 2 stops, ranks them by distance, compares to the great-circle line, and surfaces the world's most-used transfer hubs.
`Python` · `pandas` · `NumPy` · `Matplotlib`

**[bi](https://github.com/optmstic/bi) — Apple Financial Forecasting Dashboard**
Interactive Power BI dashboard with Azure AutoML predictions on 2006–2025 Apple financial data. Revenue, gross margin and volume forecasts integrated with dynamic KPIs.
`Power BI` · `Azure ML` · `Python` · `DAX` · `Prophet / ARIMA / LightGBM`

**[sad](https://github.com/optmstic/sad) — Seoul Bicycle-Sharing Demand Forecast**
End-to-end IBM Data Analyst capstone: web scraping + OpenWeather API, feature engineering, regularized regression with `glmnet`, and an R Shiny dashboard simulating a city planner's view of demand.
`R` · `tidyverse` · `glmnet` · `Shiny`

**[rnrd](https://github.com/optmstic/rnrd) — AdventureWorks Auction & Store Expansion**
Extension of the AdventureWorks schema: an auction subsystem (tables + stored procedures for clearance listings, bidding, auto-close) plus analytical queries recommending two US cities for the first brick-and-mortar stores.
`T-SQL` · `SQL Server` · `AdventureWorks`
